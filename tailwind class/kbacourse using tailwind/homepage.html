Asociado Certificado en Blockchain

Asociado Certificado en Blockchain (CBA) es el programa de certificación insignia ofrecido por Kerala Blockchain Academy en colaboración con MLG Blockchain, Canadá. Este curso proporciona a los nuevos aprendices un recorrido completo por los fundamentos de esta tecnología. Su objetivo principal es hacer más accesible el aprendizaje de blockchain, una tarea que puede resultar desafiante. Adaptándose a los últimos.

Hyperledger Fabric Fundamentals(Go)

Hyperledger Fabric Fundamentals focuses on the technical and logical architecture of various components in Hyperledger Fabric using fictional case study. The learners get to understand a business scenario, model a consortium network, and also write a Chaincode for the required business logic. Learners can interact with the Chaincode using Microfab, a tool to easily start the Hyperledger Fabric network fundamental.
